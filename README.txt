
SPEEDY AI 2.0 - Your Super-Intelligent Assistant

HOW TO USE:

1. INSTALL DEPENDENCIES:
   - Install Python 3.10+
   - Install Ollama: https://ollama.com/download
   - Run: ollama pull llama3
   - Run: pip install -r requirements.txt

2. START SPEEDY:
   - Run: streamlit run speedy.py
   - It will open in your browser.
   - You can chat, upload PDFs, listen to replies.

FEATURES:
✔️ Local LLaMA 3 model
✔️ Voice output
✔️ File reader (PDF/TXT)
✔️ Chat memory
✔️ Web UI (mobile-friendly)
✔️ Save conversation

This is YOUR fully offline Claude/GPT-style AI.
